# Smart_Crop_Project
We structured our project using a modular architecture where each functionality like AI detection, soil analysis, irrigation, and pest control is separated into independent service modules. The controller acts as the central unit that integrates all components and returns a final decision.

# 🌾 Smart Farming AI System

An AI-powered smart agriculture system that detects crop health issues, predicts risks, and automates farming operations using machine learning, IoT, and environmental data.

---

## 🚀 Features

### 🌱 Crop Health Detection
- Detects nutrient deficiencies (Nitrogen, Phosphorus, Potassium)
- Uses AI-based image analysis (simulated)

### 🧪 Soil Analysis
- Monitors soil moisture and nutrient levels
- Identifies deficiencies in real-time

### ☁️ Weather Forecast Integration
- Predicts environmental conditions
- Helps in planning irrigation and crop protection

### 💧 Smart Irrigation System
- Automatically controls water motor based on soil moisture

### 🌿 Greenhouse Automation
- Maintains optimal temperature and humidity
- Controls fans, heaters, and humidifiers

### 🐛 Pest Control System
- Predicts pest risks based on crop + weather data
- Suggests pesticide usage
- Supports automatic pesticide spraying
